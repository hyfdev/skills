# skills

Reusable skills for Claude Code and Codex.

## Skills

- [hyfdev-rolldown-trust-review](skills/hyfdev-rolldown-trust-review/SKILL.md) — Review Rolldown PRs with problem-first reasoning, independent adversarial review, prioritized inline findings, and evidence-backed Approve or Request changes decisions.

## Install

Clone this repository, then copy or symlink the desired skill directory into the client-specific personal skills directory.

- Claude Code: `~/.claude/skills/<skill-name>/`
- Codex: `~/.agents/skills/<skill-name>/`
